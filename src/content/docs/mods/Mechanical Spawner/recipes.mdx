## Individual Parts
### Recipe Type
`create_mechanical_spawner:spawner`
### Input
```json
{
"input": {
    "type": "fluid_stack",
    "amount": 100,
    "fluid": "create_mechanical_spawner:spawn_fluid_bat"
  },
},
```
### Output
```json
  "output": "minecraft:bat",
```

### Processing time
<CodeVariants
	json={
		`
		"processingTime": 2000
		`
	}
	kube={
		`
		.processingTime(2000)
		`
	}

/>
### Custom Loot (for loot collectors)
<CodeVariants
	json={
		`
		"customLoot": [
      {
        "id": "minecraft:nether_star"
      },
      {
        "count": 16,
        "id": "create:experience_nugget"
      },
      {
        "chance": 0.5,
        "count": 8,
        "id": "create:experience_nugget"
      }
    ],
		`
	}
	kube={
		`
		.customLoot([
      Item.of("minecraft:nether_star"),
      Output.of("16x create:experience_nugget"),
      Output.of("8x create:experience_nugget",0.5)
    ])
		`
	}

/>
