> Recipes with same mesh and input block gets merged allowing easier compat.

## Individual Parts
### Mesh
```json
{
  "mesh": {
    "count": 1,
    "id": "createsifter:sturdy_mesh"
  },
},
```
### Processing time
<CodeVariants
	json={
		`
		  "processingTime": 500,
		`
	}
	kube={
		`
		.processingTime(500)
		`
	}

/>
### Waterlogged (optional)
- Recipe requires brass sifter.

<CodeVariants
	json={
		`
		"waterlogged": "true"
		`
	}
	kube={
		`
		.waterlogged(true)
		`
	}

 />

### Advanced sifter (brass) per recipe (optional)
- Recipe requires brass sifter.

<CodeVariants
	json={
		`
		"advancedSifter": "true"
		`
	}
	kube={
		`
		.advancedSifter(true)
		`
	}

 />

### Recipe Requirements
> See [Mechianicals Recipe Requirements](/mods/mechanicals-lib/recipe-requirements/)

### Compact recipes.
#### Current mods:
  - EnderIO
  - Ars nouveau
  - AE2

## Examples
### Basic
<CodeVariants
	json={
		`
		{
			"type": "createsifter:sifting",
			"input": {
				"item": "createsifter:dust"
			},
			"mesh": {
				"count": 1,
				"id": "createsifter:andesite_mesh"
			},
			"processingTime": 500,
			"results": [
				{
					"chance": 0.2,
					"count": 2,
					"id": "minecraft:redstone"
				},
				{
					"chance": 0.1,
					"id": "minecraft:glowstone_dust"
				},
				{
					"chance": 0.4,
					"id": "minecraft:bone_meal"
				},
				{
					"chance": 0.01,
					"id": "minecraft:blaze_powder"
				},
				{
					"chance": 0.2,
					"id": "create:experience_nugget"
				}
			]
		}
		`
	}
	kube={
		`
			event.recipes.createsifter.sifting(
				[
					Output.of('2xminecraft:redstone',0.2),
					Output.of('minecraft:glowstone_dust',0.1),
					Output.of('minecraft:bone_meal',0.4),
					Output.of('minecraft:blaze_powder',0.01),
					Output.of('create:experience_nugget',0.2),
				],'createsifter:dust',"createsifter:andesite_mesh"),
		`
	}

/>
